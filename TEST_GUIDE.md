# 医院终端优化工具 - 测试指南

## 测试环境要求
- Windows 7 SP1 或更高版本
- .NET Framework 4.6.2+
- 医院域网环境（可选，用于权限测试）
- 管理员权限（某些测试需要）

## 性能测试

### 测试23: 启动时间
**测试方法：**
```bash
@echo off
setlocal enabledelayedexpansion
for /f "tokens=1-4 delims=:.," %%a in ("%time%") do (
   set /a start=%%a*3600000+%%b*60000+%%c*1000+%%d*10
)
HospitalTerminalOpt.exe
for /f "tokens=1-4 delims=:.," %%a in ("%time%") do (
   set /a end=%%a*3600000+%%b*60000+%%c*1000+%%d*10
)
set /a elapsed=!end!-!start!
echo 启动时间: !elapsed! 毫秒
```
**预期结果：**
- 启动时间 < 1000毫秒（1秒）

## 故障排除

### 应用无法启动
- 检查.NET Framework 4.6.2+是否安装
- 检查管理员权限
- 查看审计日志中的错误信息

### 医疗设备无法识别
- 检查WMI服务是否正常运行
- 验证设备驱动是否已安装
- 查看设备检查日志

### 权限识别错误
- 检查Windows域组配置
- 确认当前用户是否在相应的AD组中
- 尝试重新登录

### 配置文件错误
- 删除config目录重新生成
- 检查目录权限是否正确
- 查看审计日志获取更多信息
