========================================
医院终端系统优化工具 - 快速开始指南
========================================

【重要】首次使用前必读

1. 系统要求检查
   - Windows 7 SP1 或更高版本
   - .NET Framework 4.6.2+ 已安装
   - 至少50MB磁盘空间

   检查.NET版本方法：
   按 Win+R，输入：cmd
   运行：reg query "HKLM\SOFTWARE\Microsoft\NET Framework Setup\NDP\v4\Full" /v Release
   
   如果提示找不到，需要安装.NET Framework 4.6.2：
   下载地址：https://www.microsoft.com/zh-cn/download/details.aspx?id=53344

2. 安装步骤（3步完成）
   
   第一步：解压文件
   - 将本部署包解压到医疗终端（推荐位置：D:\HospitalTools）
   
   第二步：首次启动
   - 双击 HospitalTerminalOpt.exe
   - 首次运行会自动创建 config 和 logs 目录
   - 系统会自动检测医疗设备
   
   第三步：验证安装
   - 界面显示用户身份和系统状态
   - 医疗设备识别成功
   - 可以点击各按钮执行操作

3. 基本操作（3种方式）

   【方式1】前台运维（有界面）
   双击 HospitalTerminalOpt.exe
   - 显示图形界面
   - 可点击按钮执行各项操作
   - 实时查看执行结果
   - 适合运维人员日常维护

   【方式2】快速优化（后台无界面）
   在命令行运行：
   HospitalTerminalOpt.exe -silent -optimize quick
   - 清理临时文件（约5分钟）
   - 无UI显示
   - 结果保存到日志
   - 适合业务时间快速优化

   【方式3】深度优化（后台无界面，夜间推荐）
   在命令行运行：
   HospitalTerminalOpt.exe -silent -optimize deep
   - 完整系统优化（约30分钟）
   - 无UI显示
   - 业务高峰自动禁用
   - 适合夜间自动运行

4. 配置自动优化（可选）

   Windows 10/11：
   - 按 Win+R
   - 输入：taskschd.msc
   - 右键"任务计划程序库"
   - 选择"创建基本任务"
   - 名称：医院终端夜间优化
   - 触发器：每天 22:30
   - 操作：运行程序
   - 程序：D:\HospitalTools\HospitalTerminalOpt.exe
   - 参数：-silent -optimize deep

5. 常见问题快速解决

   Q1: 双击无反应
   A: 按 Win+R，输入 cmd 打开命令行
      切换到exe所在目录：cd /d D:\HospitalTools
      运行：HospitalTerminalOpt.exe
      查看是否有错误提示

   Q2: 提示缺少.NET Framework
   A: 访问链接下载并安装：
      https://www.microsoft.com/zh-cn/download/details.aspx?id=53344
      安装后重启电脑

   Q3: 医疗设备无法识别
   A: 在命令行运行：net start winmgmt
      这会启动WMI服务（设备检查依赖）

   Q4: 无法创建config目录
   A: 检查文件夹权限
      在exe所在文件夹右键 > 属性 > 安全
      确保当前用户有"修改"权限

6. 文件说明

   HospitalTerminalOpt.exe     - 主程序（单文件）
   config/                     - 配置目录（自动生成）
   logs/                       - 日志目录（自动生成）
   PRECOMPILED_PACKAGE.md      - 详细部署指南
   README.md                   - 功能说明
   DEPLOYMENT_GUIDE.md         - 部署方案

7. 重要提示

   ⚠ 不要删除 config 和 logs 目录
   ⚠ 不要修改 exe 文件
   ⚠ 定期备份 logs 目录（包含重要审计记录）
   ⚠ 不要在医疗业务时间执行深度优化（系统会自动禁用）

8. 获得帮助

   遇到问题时：
   1. 查看 logs/ 目录中的错误日志
   2. 运行 HospitalTerminalOpt.exe -help 查看帮助
   3. 参考 PRECOMPILED_PACKAGE.md 中的故障排除
   4. 联系IT运维部门
